# HOPF-OmniAtlas

**UPRS Möbius–Hopf OmniAtlas** — автономное одностраничное научное приложение
(`index.html`, без зависимостей и сборки): атлас вычислимых редуцированных
физических моделей, отображаемых на диаграмму Смита, сферу Римана S², трисферу S³,
хопфионы и спектральные 4D-проекции.

*A self-contained single-file scientific workbench: an atlas of computable reduced
physical models rendered on the Smith chart, Riemann sphere S², trisphere S³,
Hopf fibrations and 4D spectral projections. Open `index.html` in a browser — that is all.*

## Запуск

Откройте `index.html` в современном браузере. Всё — модели, решатели, рендер,
самотесты — работает локально, без сети.

## v3.9.8 · Предиктивная обсерватория феноменов

Замкнутый контур предсказательной работы **прогноз → проверка → калибровка**
(кнопка «🔭 Обсерватория» в верхней панели):

- **Детекторы феноменов** на активном расчётном локусе: резонансные минимумы и
  максимумы |Γ| с субсеточным уточнением и оценкой добротности Q, полные обходы
  фазы (2π·k), пересечения границы пассивности |Γ|=1, аномалии группового
  запаздывания (робастная z-оценка по медиане/MAD), каспоподобные изломы локуса.
- **Лестницы феноменов**: обнаруженные позиции проверяются на арифметическую и
  геометрическую прогрессию; следующие ступени экстраполируются за пределы
  расчётного диапазона со спадающей достоверностью (статус `EXTRAPOLATED`).
- **Параметрические законы**: сканирование параметров модели, подгонка степенного
  закона x*(p) ≈ A·p^α с R² и leave-one-out-валидацией; прогноз положения
  феномена при ещё не рассчитанных значениях параметров (`MODEL-PREDICTED`).
- **Пороги рождения/исчезновения феноменов**: бисекция критического значения
  параметра, при котором меняется число наблюдаемых явлений (`HYPOTHESIS`).
- **Фальсифицируемые протоколы**: каждый прогноз содержит точный численный тест
  (окно, допуск, целевое значение) и может быть проверен одним нажатием —
  локальным пересчётом модели с высоким разрешением.
- **Журнал прогнозов и самокалибровка**: результаты проверок накапливаются
  (localStorage), считаются точность попаданий и показатель Брайера; фактор
  калибровки автоматически ужесточает или ослабляет достоверность будущих
  прогнозов. Экспорт полного пакета — JSON/CSV с происхождением данных.

### Научная граница

Все «подтверждения» обсерватории — численные, в рамках выбранной модели и её
области применимости; они не заменяют эксперимент. Прогнозы вне расчётного
диапазона всегда маркируются `EXTRAPOLATED`, `MODEL-PREDICTED` или `HYPOTHESIS`.

*All observatory "confirmations" are numerical, within the selected model and its
validity domain; they do not replace experiment.*

## Самотесты

Кнопка «Самотест» в панели параметров (или `runTests(true)` в консоли) прогоняет
сквозной набор проверок всех модулей, включая v3.9.8: точность локализации
резонанса RLC (f₀ = 1/(2π√LC)), непрерывность развёртки фазы, экстраполяцию
лестниц, восстановление показателя степенного закона, цикл численной верификации
и границы калибровочных метрик.

## API

- `__UPRS398__.scanDeep()` / `.verifyAll()` / `.audit()` / `.exportJSON()` —
  программный доступ к обсерватории;
- `__UPRS398__.core` — чистые функции (детекторы, лестницы, подгонка законов,
  верификация) для встраивания и тестирования.
